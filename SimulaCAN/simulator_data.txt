### 1. Vehicle Total Mileage (Km)

The standard PGN for total distance (PGN 65248) is not in your list. However, the document specifies a high-resolution version which is present in your log.

*   **Parameter:** High Resolution Total Vehicle Distance
*   **PGN:** 65217 (0xFEC1)
*   **SPN:** 917
*   **Frame ID from your log:** `0x18FEC1EE`
*   **Data Payload Location:**
    *   **Bytes:** 1-4 (4 bytes, Little Endian)
*   **Conversion:**
    *   **Resolution/Scaling:** 5 m/bit
    *   **Offset:** 0
    *   **Formula:** `Total Distance (km) = (Raw_Value * 5) / 1000`
*   **For your simulator:** To inject a value, you would calculate the desired raw value (`Desired_km * 1000 / 5`), convert it to a 32-bit integer, and place it into bytes 1, 2, 3, and 4 (LSB first) of any frame with ID `0x18FEC1EE`.

### 2. Fuel Tank Level (%)

*   **SPN:** 96
*   **Frame ID from your log:** `0x18FEFC47`
*   **Data Payload Location:**
    *   **Byte:** 2 (1 byte)
*   **Conversion:**
    *   **Resolution/Scaling:** 0.4 %/bit
    *   **Offset:** 0
    *   **Formula:** `Fuel Level (%) = Raw_Value * 0.4`
*   **For your simulator:** To inject a value of 14.8%, you would calculate the raw value (`14.8 / 0.4 = 37`), and place this value (0x25) into byte 2 of any frame with ID `0x18FEFC47`.

### 3. Vehicle Speed (Km/h)

This is a fundamental vehicle parameter, broadcast frequently.

*   **Parameter:** Wheel-Based Vehicle Speed
*   **PGN:** 65265 (0xFEF1)
*   **SPN:** 84
*   **Frame ID from your log:** `0x18FEF100`
*   **Data Payload Location:**
    *   **Bytes:** 2-3 (2 bytes, Little Endian)
*   **Conversion:**
    *   **Resolution/Scaling:** 1/256 km/h per bit (approx. 0.00390625 km/h per bit)
    *   **Offset:** 0
    *   **Formula:** `Speed (km/h) = Raw_Value * (1/256)`
*   **For your simulator:** To inject a value of 11 km/h, you would calculate the raw value (`11 * 256 = 2816`), convert it to a 16-bit integer (0x0B00), and place it into bytes 2 and 3 (LSB first) of any frame with ID `0x18FEF100`.

### 4. Engine RPM

This is one of the most common parameters, found in the primary engine controller message.

*   **Parameter:** Engine Speed
*   **PGN:** 61444 (0xF004)
*   **SPN:** 190
*   **Frame ID from your log:** `0x0CF00400`
*   **Data Payload Location:**
    *   **Bytes:** 4-5 (2 bytes, Little Endian)
*   **Conversion:**
    *   **Resolution/Scaling:** 0.125 rpm/bit
    *   **Offset:** 0
    *   **Formula:** `Engine Speed (RPM) = Raw_Value * 0.125`
*   **For your simulator:** To inject a value of 1183 RPM, you would calculate the raw value (`1183 / 0.125 = 9464`), convert it to a 16-bit integer (0x24F8), and place it into bytes 4 and 5 (LSB first) of any frame with ID `0x0CF00400`.

### 7. Engine Torque (%)

This value, expressed as a percentage, corresponds to the actual engine torque relative to a reference torque.

*   **Parameter:** Actual Engine - Percent Torque
*   **PGN:** 61444 (0xF004)
*   **SPN:** 513
*   **Frame ID from your log:** `0x0CF00400`
*   **Data Payload Location:**
    *   **Byte:** 3 (1 byte)
*   **Conversion:**
    *   **Resolution/Scaling:** 1 %/bit
    *   **Offset:** -125 %
    *   **Formula:** `Torque (%) = Raw_Value - 125`
*   **For your simulator:** To inject a value of 25%, you would calculate the raw value (`25 + 125 = 150`), and place this value (0x96) into byte 3 of any frame with ID `0x0CF00400`.
